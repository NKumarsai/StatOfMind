This repository includes files and scripts used to generate data for the "Using sentiment analysis to predict ratings of popular tv series" StatOfMInd blog post.

get_epsiode_transcripts.py: This Python script will download the episode transcripts for all seasons of any given TV shows. The episode transcripts are obtained from www.springfieldspringfield.co.uk. Here, it has been set to the "Big Bang Theory" show, but this can be replaced with any user-defined show.
